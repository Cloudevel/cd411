## Requisitos

- Cuenta en [GitHub](https://github.com)
- *Git* 2.x o superior

## Ejecución

### En GitHub Codespaces (recomendado)

Haz clic en el botón **Open in GitHub Codespaces**. El entorno se configura automáticamente con Jupyter Lab y el kernel de bash.

### En local

```bash
pip install jupyterlab bash_kernel
python -m bash_kernel.install
jupyter lab
```
